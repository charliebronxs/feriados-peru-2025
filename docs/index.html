🇵🇪 Feriados Perú 2025

Agrega los feriados oficiales peruanos a tu calendario con un solo clic.

📤 Descargar el archivo .ics

📥 Descargar .ics

📱 iPhone / iOS

Ve a Ajustes → Calendario → Cuentas → Añadir cuenta
Selecciona Otra → Añadir calendario suscrito
Pega este enlace:
https://raw.githubusercontent.com/charliebronxs/feriados-peru-2025/main/Feriados_Peru_2025_Nombrado.ics

📅 Google Calendar (Android o Web)

Entra a Google Calendar
En el panel izquierdo, haz clic en + Otros calendarios → Desde URL
Pega este mismo enlace:
https://raw.githubusercontent.com/charliebronxs/feriados-peru-2025/main/Feriados_Peru_2025_Nombrado.ics

💻 Outlook Web

Haz clic en Agregar calendario → Suscribirse desde la web
Pega el enlace y dale un nombre como "Feriados Perú 2025"

🔄 Actualización automática

Cada vez que edites el archivo feriados.csv, el archivo .ics se regenera solo gracias a GitHub Actions.
